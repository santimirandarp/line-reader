@santimir/line-reader

line-reader

Reads lines from a file or a string.

  • Class keeps track of the current index,
  • provides some helper methods.

Installation

 
npm i linereader

Examples
  • Splitting and reading a string ```javascript import {LineReader} from "line-reader"; const myString = "hello\n world\n new string\n here"; let lines = new LineReader(myString);//does not touch original string

const firstLine = lines.readLine(); // hello const secondLine = lines.readLine(); // world const nextLines = lines.readLines(2); //["new string", "here"]

const index = lines.index; //3

const getArrayBackIfNeeded = lines.data;
